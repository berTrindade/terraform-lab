# Terraform Lab

Hands-on Terraform learning through practical proof-of-concept demonstrations.

## POC Categories

### [Manage Sensitive Data](manage-sensitive-data/)
Techniques for handling secrets in Terraform - comparing ephemeral values (1.10+) vs traditional sensitive variables.

## Usage

```bash
cd manage-sensitive-data
./demo.sh
```

## Requirements

- Terraform >= 1.0
- jq (optional, for demo scripts)
